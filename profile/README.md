### Core

- **[unison-brain](https://github.com/unison-labs-ai/unison-brain)** — CLI, SDK & MCP for shared agent memory.
- **[unison-fs](https://github.com/unison-labs-ai/unison-fs)** — mount the brain as a filesystem (FUSE / NFS).

### Tooling

- **[install-mcp](https://github.com/unison-labs-ai/install-mcp)** — one command to install the Unison MCP into any client.
- **[code-chunk](https://github.com/unison-labs-ai/code-chunk)** — AST-aware code chunking for retrieval.
- **[python-sdk](https://github.com/unison-labs-ai/python-sdk)** — the brain, from Python.
